HTML Tags Table

<!DOCTYPE html> - Declares the document type and version of HTML.
<html> - The root element of the HTML document.
<head> - Contains meta-information about the document.
<meta> - Specifies character set and viewport settings.
<title> - Sets the title of the webpage displayed in the browser tab.
<style> - Contains internal CSS styles for the document.
<body> - Contains the content of the webpage.
<h1> - Defines a top-level heading.
<h2> - Defines a second-level heading.
<table> - Defines a table.
<tr> - Defines a table row.
<td> - Defines a table data cell.
<th> - Defines a table header cell.
<a> - Defines a hyperlink.
<p> - Defines a paragraph.
<div> - Defines a division or section in the document.
<ul> - Defines an unordered list.
<li> - Defines a list item.
<footer> - Represents the footer section of the document.
